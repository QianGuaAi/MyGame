# 王冠前哨

塔防 + 三英雄装备养成原型。

当前主开发版本是 Phaser/Vite 版，代码在 `src/`。Godot 文件暂时作为迁移参考保留。

## 运行

```powershell
npm install
npm run dev -- --port 5173
```

打开：

[http://127.0.0.1:5173/](http://127.0.0.1:5173/)

## 构建

```powershell
npm run build
```

## 重要文档

- `doc/game-design.md`：游戏设定来源。
- `doc/development-guide.md`：开发协作方式。
- `AGENTS.md`：给 AI/自动化协作者的项目说明。
