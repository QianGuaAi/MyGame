# 游戏设定文档

## 游戏核心

本游戏是塔防 + 英雄养成 + 装备商店 + 随机商人玩法。

玩家通过关卡获得金币，用金币购买装备、强化英雄。战斗中玩家建造和升级防御塔抵挡敌人。敌人有概率掉落装备，特殊商人有概率出售特殊塔图纸。每章末尾会出现精英商人，提供更高级的装备和图纸。

## 货币与资源

金币用于购买英雄装备、商人商品和特殊物品。

建议后续可以把战斗建塔资源和英雄装备金币分开：

- 金币：用于装备、商人、彩蛋奖励。
- 建造资源：用于战斗内建塔和升级塔。

如果暂时不拆分资源，也可以先统一使用金币。

## 英雄设定

游戏中有 3 个英雄：

- 王铁柱
- 王二狗
- 白月光

### 王铁柱

定位：前排防御英雄。

特点：

- 高血量
- 高力量
- 高护甲
- 适合装备锤、铁甲、盾

重要规则：

- 王铁柱没有嘲讽技能。
- 格挡是王铁柱的被动技能。

被动技能：格挡

受到攻击时有概率格挡，减少本次受到的伤害。

| 阶段 | 格挡率 | 格挡效果 |
|---|---:|---:|
| 初始 | 15% | 减少 40% 伤害 |
| 中期升级 | 20% | 减少 45% 伤害 |
| 后期升级 | 25% | 减少 50% 伤害 |

可选技能方向：

| 技能 | 类型 | 效果 |
|---|---|---|
| 格挡 | 被动 | 概率减少受到的伤害 |
| 重击 | 主动 | 对单个敌人造成伤害并短暂眩晕 |
| 铁骨 | 被动 | 生命值低于 30% 时护甲提升 |
| 破甲锤 | 主动 | 攻击精英或 Boss 时降低其护甲 |

### 王二狗

定位：灵活输出英雄。

特点：

- 攻速较快
- 暴击能力强
- 适合补刀
- 可以作为对空输出角色

推荐装备：

- 矛
- 剑
- 弓
- 锁甲

### 白月光

定位：法术辅助英雄。

特点：

- 法术伤害
- 治疗
- 减速
- 强化附近塔或英雄

推荐装备：

- 基础法杖
- 布甲
- 藤甲

## 英雄基础力量建议

| 英雄 | 基础力量 | 说明 |
|---|---:|---|
| 王铁柱 | 7 | 能装备铁甲、盾、重型武器 |
| 王二狗 | 5 | 能装备锁甲、盾，但不如王铁柱耐打 |
| 白月光 | 3 | 不能装备力量要求较高的防具 |

## 装备商店

装备商店使用金币兑换装备。

| 装备 | 价格 | 效果 | 限制 |
|---|---:|---|---|
| 基础法杖 | 7 | 法术伤害 +6，技能冷却 -5% | 适合法术英雄 |
| 矛 | 8 | 攻击 +8，近战攻击距离略微增加 | 无 |
| 锤 | 15 | 攻击 +14，10% 概率眩晕 | 建议力量要求 6 |
| 剑 | 10 | 攻击 +10，暴击率 +8% | 无 |
| 弓 | 6 | 攻击 +6，可攻击飞行敌人，攻速 +10% | 无 |
| 布甲 | 5 | 生命 +15，法抗 +2 | 无 |
| 锁甲 | 12 | 生命 +25，护甲 +4 | 力量要求 5 |
| 藤甲 | 9 | 生命 +20，每秒少量回血，受到火焰伤害增加 | 无 |
| 铁甲 | 18 | 生命 +40，护甲 +7，移速 -8% | 建议力量要求 7 |
| 盾 | 14 | 格挡率 +15% | 力量要求 5 |

## 装备品质

装备可以分为 4 个品质：

| 品质 | 说明 |
|---|---|
| 普通 | 基础商店常见装备 |
| 稀有 | 敌人低概率掉落，野生商人可能出售 |
| 史诗 | 精英敌人、Boss 或高级商人产出 |
| 传说 | 章节 Boss、精英商人产出 |

传说装备不建议让普通敌人直接掉落过多，否则会破坏后期经济和成长节奏。

## 敌人掉落

稀有敌人和精英敌人有概率爆装备。

推荐掉率：

| 敌人类型 | 掉落概率 |
|---|---:|
| 普通敌人 | 1% 到 3% |
| 稀有敌人 | 8% 到 12% |
| 精英敌人 | 12% 到 20% |
| 小 Boss | 35% |
| 章节 Boss | 必掉稀有以上装备或图纸碎片 |

掉落内容可以包括：

- 普通装备
- 稀有装备
- 图纸碎片
- 少量金币
- 特殊消耗品

## 野生商人

每三关会出现一个野生商人。

出现规则：

- 第 3 关、第 6 关、第 9 关等关卡结束后出现。
- 玩家打完该关后，可以向野生商人购买特殊东西。
- 野生商人有概率出售特殊塔图纸。

野生商人商品建议：

| 商品类型 | 出现概率 |
|---|---:|
| 回复药、临时增益 | 高 |
| 打折普通装备 | 中 |
| 稀有装备 | 中 |
| 图纸碎片 | 40% |
| 特殊塔图纸 | 25% |

野生商人可出售的特殊物品例子：

- 回血药
- 攻速药
- 临时护盾
- 随机稀有装备
- 特殊塔图纸
- 图纸碎片

## 精英商人

每一章最后会出现精英商人。

出现规则：

- 每章最后一关结束后出现。
- 会出现传说级装备和武器。
- 玩家只能购买 2 个商品。
- 这 2 个商品可以给 2 个英雄装备。
- 精英商人必定出现特殊塔图纸。

商品规则建议：

- 必定出现 1 张特殊塔图纸。
- 必定出现 2 到 3 件传说装备。
- 玩家最多购买 2 件商品。
- 至少有 1 件商品适合当前队伍，避免全部不可用。

传说装备例子：

| 装备 | 效果 |
|---|---|
| 破山锤 | 高攻击，眩晕概率提升，对 Boss 额外伤害 |
| 月影弓 | 攻速提升，攻击有概率连射 |
| 星辉法杖 | 白月光治疗增强，附近塔攻速提升 |
| 不动铁甲 | 大幅加血，受到连续攻击时获得护盾 |
| 将军盾 | 格挡率提升，格挡成功时反伤 |

注意：将军盾不能提供嘲讽效果，因为王铁柱的嘲讽设定已经取消。

## 特殊塔图纸

特殊塔图纸可以通过野生商人、精英商人、Boss 掉落或图纸碎片合成获得。

建议规则：

- 获得图纸后永久解锁对应特殊塔。
- 图纸碎片可以合成完整图纸。
- 精英商人必定出售至少 1 张特殊塔图纸。

特殊塔例子：

| 特殊塔 | 作用 |
|---|---|
| 冰霜塔 | 范围减速，适合控场 |
| 火焰塔 | 持续灼烧，克制回血敌人 |
| 雷电塔 | 连锁攻击，适合密集敌人 |
| 毒藤塔 | 持续伤害，越打越痛 |
| 祭坛塔 | 给附近英雄或塔加攻速 |
| 金币塔 | 击杀敌人后额外金币，但伤害较低 |

## 彩蛋

一些彩蛋会提供金币，但是每个彩蛋只能出现一次。

彩蛋规则：

- 每个彩蛋只能触发一次。
- 触发后记录为已获得。
- 彩蛋奖励以金币为主，也可以少量给图纸碎片。

彩蛋例子：

| 彩蛋 | 奖励 |
|---|---:|
| 点击地图角落的罐子 | 金币 +20 |
| 王二狗靠近隐藏草丛 | 金币 +30 |
| 白月光站在月光祭坛旁 | 图纸碎片 +1 |
| 王铁柱连续格挡多次 | 金币 +50 |

## 弓箭塔

弓箭塔是基础远程塔，前三级为固定成长，四级开始分支升级。

| 等级 | 伤害 | 攻速 | 说明 |
|---|---:|---:|---|
| 一级 | 15 | 1.0 | 基础远程塔 |
| 二级 | 20 | 1.1 | 稳定提升 |
| 三级 | 25 | 1.2 | 解锁四级分支 |
| 四级：爆裂箭塔 | 40 | 1.5 | 群体伤害，适合清小怪 |
| 四级：鹰眼弩塔 | 30 | 2.5 | 单体高频输出，适合打 Boss 和精英 |

四级分支说明：

- 爆裂箭塔：范围伤害路线，适合处理大量小怪。
- 鹰眼弩塔：单体高速路线，适合处理高血量敌人、精英敌人和 Boss。

数值理解：

- 爆裂箭塔单体 DPS 为 60，并且带群体伤害。
- 鹰眼弩塔单体 DPS 为 75，单体输出更强。

## 关卡节奏建议

每 3 关形成一个小循环：

| 关卡 | 内容 |
|---|---|
| 第 1 关 | 基础敌人，熟悉建塔 |
| 第 2 关 | 出现较硬敌人 |
| 第 3 关 | 关后出现野生商人 |
| 第 4 到 5 关 | 提升敌人强度，可能出现稀有敌人 |
| 第 6 关 | 关后再次出现野生商人 |
| 第 9 关 | 关后出现野生商人，可提高图纸概率 |
| 每章最后 | 出现精英商人，必定有特殊塔图纸和传说装备 |

## 当前已确认的关键修改

- 王铁柱不再拥有嘲讽技能。
- 王铁柱的格挡是被动技能。
- 装备商店包含基础法杖、矛、锤、剑、弓、布甲、锁甲、藤甲、铁甲、盾。
- 锁甲需要力量 5。
- 盾需要力量 5。
- 稀有敌人有概率掉装备。
- 每三关出现野生商人。
- 野生商人有概率出售特殊塔图纸。
- 每章最后出现精英商人。
- 精英商人会出售传说级装备武器。
- 精英商人只能购买 2 个商品。
- 精英商人必定出现特殊塔图纸。
- 彩蛋可以提供金币，但只能出现一次。
- 弓箭塔四级分为群体伤害路线和单体高速路线。
